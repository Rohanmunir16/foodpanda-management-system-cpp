# FoodPanda Management System (C++)

A console-based FoodPanda Management System developed in C++.  
This project simulates a food ordering and delivery management system using linked lists, graphs, and basic data structures.

---

## Features

- Purchase Products
- Check All Orders
- Check Bills
- Update Orders
- Cancel Orders
- Deliver Orders
- View Delivered Orders
- Customer Feedback Section
- Distance Calculation Between Sectors
- Input Validation for Name and Phone Number

---

## Technologies Used

- C++
- Linked Lists
- Graph Representation
- Arrays
- Vectors
- Console-based UI
